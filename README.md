# Sylvia-Exe.github.io
Website for stormworks in game company called Odyssey

## Local files

- `index.html` - main webpage
- `styles.css` - styling for the webpage

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. On GitHub, open **Settings** for this repository.
3. Open **Pages** in the left sidebar.
4. Under **Build and deployment**:
	- Source: **Deploy from a branch**
	- Branch: **main** (or your default branch)
	- Folder: **/ (root)**
5. Click **Save**.
6. Wait about 1-2 minutes, then open:
	- `https://<your-github-username>.github.io/`

For a repository named `<your-github-username>.github.io`, GitHub serves `index.html` at the site root automatically.
